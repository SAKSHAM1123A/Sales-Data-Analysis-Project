# 📊 Sales Data Analysis using Python & Pandas

A beginner-friendly **Sales Data Analysis** project built using **Python** and **Pandas**. This project demonstrates how to load, clean, analyze, and generate meaningful business insights from sales data stored in a CSV file.

---

## 📌 Project Overview

The purpose of this project is to analyze sales data and generate useful business insights by performing data cleaning, data exploration, and statistical analysis.

The program:
- Loads sales data from a CSV file.
- Cleans missing and duplicate values.
- Calculates key business metrics.
- Finds the best-selling product.
- Identifies the top-performing region.
- Generates a formatted sales report.

---

## 🚀 Features

- 📂 Load CSV data using Pandas
- 🧹 Data Cleaning
  - Remove duplicate records
  - Handle missing values
- 📈 Calculate important sales metrics
- 🏆 Identify the best-selling product
- 🌍 Find the top-performing sales region
- 📝 Automatically generate a sales report (.txt)
- 💻 Clean and well-commented Python code

## ▶️ Running the Project

Run the following command:

```bash

```

After execution, the program will:

- Load the dataset
- Clean the data
- Analyze sales
- Display insights
- Generate `sales_report.txt`
